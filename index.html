First Steps with AngularJS September 5th, 2014

About AngularJS

  • AngularJS is an open-source JavaScript framework for building web apps.
  • Maintained by Google and the open-source community.
  • Like other JavaScript frameworks (you might have heard of Backbone, Ember or React), it tries to make life easier for web developers.
  • From the website, it's "HTML enhanced for web apps!" This refers to the way Angular integrates with your HTML. Angular adds special ng- attributes to your DOM elements that give them new abilities. A major concept of Angular is data-binding, which keeps data synced between your controller and view. More on that in a few.

Why you should consider learning AngularJS

  • Just a few lines of Angular can replace a lot of jQuery and cut down on development time.
  • Using a JavaScript framework like Angular can help you organize your JavaScript so that it's maintainable down the road—i.e., not a pile of jQuery spaghetti.
  • It's growing fast and really starting to gain traction.

Follow Along

  • If you have the git command installed, grab the repository like this:

    git clone https://github.com/ndreynolds/angular-demo

  • If not, no problem, you can grab a zipfile with the working examples here:

    https://github.com/ndreynolds/angular-demo/archive/master.zip

  • Get started by opening this page (angular-demo/index.html) in your favorite browser*. The examples are self-contained, so you shouldn't need to install anything.
*Unless your favorite browser happens to be IE<9.
